# Sentos AI Bot

AI destekli müşteri soru analiz ve taslak cevap üretim paneli.

## Özellikler
- Ürün eşleştirme
- Risk kelime tespiti
- Kurumsal dilde taslak üretimi
- Streamlit arayüz

## Çalıştırma

pip install streamlit openai pandas openpyxl

python -m streamlit run app.py
